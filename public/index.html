Talking

Visual Perception and Communication in Humans and Machines

Talking is an artistic research project which explores the mechanics of perception and conversation. Based on an analysis of human interaction, we create pairs of machines which embody specific aspects of visual communication.

We've developed two parts of this project so far, each using the same setup: Two identical devices, consisting of camera, computer, and screen, are positioned facing one another, allowing them to see and talk to each other.
